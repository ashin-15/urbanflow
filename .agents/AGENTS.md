# Workspace Rules & Context — UrbanFlow

## Dependencies and Requirements
- Always keep dependencies updated in `pyproject.toml` when introducing new Python libraries or modules.
- Run setup or installation commands when introducing new packages to keep the environment synchronized.
- Document any external module additions.

## Git Commits
- Make clean, descriptive Git commits after significant milestones or big changes are made to the codebase (e.g., completing preprocessing, feature engineering, training baselines, hyperparameter tuning, or dashboard setup).
